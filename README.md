# Suraagh — Lost & Found Mobile Application

## Description
Suraagh is an Android application that enables users to report lost and found items, manage posts, and communicate securely. It uses Firebase services for authentication, data storage, and real-time features.

## Tech Stack
- **Frontend:** Android (Java, XML)
- **Backend:** Firebase (Authentication, Firestore, Realtime DB, FCM), Node.js (Vercel)
- **Database:** Firebase Firestore
- **Storage:** Firebase Storage

## Features
- User authentication (signup, login, password reset)
- User profile management
- Post management dashboard
- Local data caching (SharedPreferences)
